# Калькулятор оценок

Простой инструмент для расчёта средних оценок и планирования успеваемости.

## Функциональные возможности

- **Ввод оценок**  
  Поле для ввода числовых значений через запятую или пробел (например, `4, 5, 3` или `4 5 3`).

- **Расчёт среднего**  
  Кнопка, запускающая вычисление среднего арифметического введённых оценок.

- **Прогноз оценки**  
  Возможность задать желаемое среднее значение и количество будущих работ, чтобы рассчитать, какие оценки необходимо получить.

- **История расчётов**  
  Отображение предыдущих результатов.
